[[continuous PDF]] of a [[discrete random variable]] with the same probability of taking all values inside an interval $[a,b]$ and 0 for all other values.
## probability function
$$
f_X(X = x)=
\begin{cases}
\frac{1}{b-a}
,& \text{if } x \in [a,b]\\
0
,& \text{otherwise}
\end{cases}
$$
