### loss function
- the [[loss function]] is a non-negative [[function]] $L: \mathcal{Y} \times \mathcal{\hat{Y}} \to [0, \infty]$ that measures how much the model output $f(x_i)=\hat{y_i}$ differs from the label $y$

### Examples
- [[mean square error]]
- [[mean absolute error]]
- [[cross entropy]]
- [[likelihood function]]
