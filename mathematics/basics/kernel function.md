### kernel function
- weighting function for a weighted sum or [[integral]]
- [[kernel function]] $K$ can be written as follows based an [[inner product]] $\langle . ,. \rangle_\mathcal{V}$ and a feature map $\varphi: \mathcal{X} \to \mathcal{V}$
$$
K(X, Y) = \langle \varphi(X), \varphi(Y) \rangle_\mathcal{V}
$$
