### power set
- given a set $A$ the [[power set]] $\mathcal{P}(S)$ is a set that contains all subsets of $S$

$$
\mathcal{P}(S) = \{A \mid A \subseteq S \}
$$
- [[cardinality]] of a [[power set]]:
$$
\left|\mathcal{P}(S) \right| = 2^{\left|S\right|} 
$$
