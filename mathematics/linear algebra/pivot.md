## Definition [[pivot]]
An elelemt $a_{ij}$ if m [[matrix]] $A$ is a [[pivot]] element if the following conditions are true
1) $a_{ij} \neq 0$
2) all elements below $a_{ij}$ are zero and all elements on the right of  $a_{ij}$ are zero $\forall k \geq i, l \leq j,(kl) \neq (ij) :a_{kl} = 0$


## examples

$$
\begin{split}
A = \begin{pmatrix}
a_{(1:1)} &\ a_{(1:2)} &\ 0 \\
a_{(2:1)} &\ a_{(2:2)} &\ a_{(2:3)} \\
0		&\ a_{(3:2)} &\ a_{(3:3)} \\  
		0		 &0 &\ 0 \\  
\end{pmatrix}
\end{split}
$$
$a_{(2:1)}$ and $a_{(3:2)}$ are [[pivot]] elements


$$
\begin{split}
A = \begin{pmatrix}
0 &\ a_{(1:2)} &\ 0 \\
a_{(2:1)} &\ 0 &\ a_{(2:3)} \\
\end{pmatrix}
\end{split}
$$
$a_{(2:1)}$ is a pivot [[pivot]] element
