### level set
- given a [[function]] $f: X \to Y$ 
- the [[level set|level c set]] of the $f$ contains all values out of the [[domain]] of $x$ with a function value equal to $c$
$$
N_c(f) = \{x \in X: f(x) = x\}
$$
