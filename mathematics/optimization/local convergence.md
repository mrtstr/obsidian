### local convergence
- if an iterative methode produces a series $\{x_n\}$ that is [[convergence|converging]] if the stating point $x_0$ is close enough to $x^*$

### global convergence
- if an iterative methode produces a series $\{x_n\}$ that is [[convergence|converging]] for an arbitrary starting point $x_0$

### convergence in a [[normed space]] $\left(V, +, \cdot, ||\cdot||\right)$
- given a [[sequence]] $v_n \in V$  
- $v_n$ is converging against $v^*$ if the following is true

$$
\begin{split}
&\forall \epsilon>0: \exists N \in \mathbb{N}: n \geq N \Rightarrow ||a_n - a^*|| \leq \epsilon \\
&\Leftrightarrow \\
&||v_n - v^* || \xrightarrow{n \rightarrow \infty} 0
\end{split}
$$
