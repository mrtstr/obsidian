## virtual private cloud
- virtual isolated network within a [[AWS]]
- is located insice a [[AWS region]] but spread over multiple [[AWS AZ (isolated datacenter)]]
- has a [[router|routing table]]
- has a [[internet protocol (IP)]] range (default is `172.31.** **0.0/16`) of [[AWS private ip]] adresses
- has multiple [[AWS subnet]] that can have a ip subspace
