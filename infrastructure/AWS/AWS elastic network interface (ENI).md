## elastic network interface
- virtual network interfaces that provide networking capabilities to [[AWS EC2]] instances
- enabling instances to connect to networks and communicate with other resources within the [[AWS]]
- per default assigned to a [[AWS private ip]] but can have assigned additional (secondary) [[AWS private ip]]
- has a [[AWS DNS hostname]] that is pointing to its [[AWS private ip]]
