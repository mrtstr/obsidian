

- piece of [networking hardware](https://en.wikipedia.org/wiki/Networking_hardware "Networking hardware") or software used
- allows data to flow from one discrete network to another.
- Gateways are distinct from [routers](https://en.wikipedia.org/wiki/Router_(computing) "Router (computing)") or [switches](https://en.wikipedia.org/wiki/Network_switch "Network switch") in that they communicate using more than one protocol to connect multiple networks[[1]]
- can operate at any of the seven layers of the [OSI model](https://en.wikipedia.org/wiki/OSI_model "OSI model").
- a connection between networks and contains devices, such as protocol translators

- On an [Internet Protocol](https://en.wikipedia.org/wiki/Internet_Protocol "Internet Protocol") (IP) network, IP packets with a destination outside a given [subnetwork](https://en.wikipedia.org/wiki/Subnetwork "Subnetwork") are sent to the network gateway
- For example, if a private network has a base [IPv4](https://en.wikipedia.org/wiki/IPv4 "IPv4") address of 192.168.1.0 and has a subnet mask of 255.255.255.0, then any data addressed to an IP address outside of 192.168.1.0–192.168.1.255 is sent to the network gateway
### Examples
For example, a network gateway connects an office or home [intranet](https://en.wikipedia.org/wiki/Intranet "Intranet") to the [Internet](https://en.wikipedia.org/wiki/Internet "Internet").
If an office or home computer user wants to load a [web page](https://en.wikipedia.org/wiki/Web_page "Web page"), at least two network gateways are accessed—one to get from the office or home network to the Internet and one to get from the Internet to the computer that serves the web page.
Ein **Internetgateway** ist eine Schlüsselstelle oder -adresse in einem lokalen [Netzwerk](https://de.wikipedia.org/wiki/Netzwerk "Netzwerk"), welche Teilnehmern innerhalb des Netzwerkes [Datenaustausch](https://de.wikipedia.org/wiki/Datenaustausch "Datenaustausch") mit dem [Internet](https://de.wikipedia.org/wiki/Internet "Internet") ermöglicht.

Als Beispiel ist hier der [Router](https://de.wikipedia.org/wiki/Router "Router") aus der privaten Nutzung zu nennen. Auf der Ebene des [IP](https://de.wikipedia.org/wiki/Internet_Protocol "Internet Protocol")-Protokolls belegt der Router die IP-Adresse z. B. 192.168.1.1. Alle an dem [Netzwerk](https://de.wikipedia.org/wiki/Netzwerk "Netzwerk") teilnehmenden PCs senden eine [Datenanfrage](https://de.wikipedia.org/w/index.php?title=Datenanfrage&action=edit&redlink=1 "Datenanfrage (Seite nicht vorhanden)") an eine [IP-Adresse](https://de.wikipedia.org/wiki/IP-Adresse "IP-Adresse"), die sich nicht im selben Netzwerk befindet (z. B. 192.168.**2**.37) an das Gerät mit der [Gateway](https://de.wikipedia.org/wiki/Gateway_(Informatik) "Gateway (Informatik)")-Adresse, also in diesem Fall den Router, welches dann die Nachricht in das übergeordnete oder in ein benachbartes [Netzwerk](https://de.wikipedia.org/wiki/Rechnernetz "Rechnernetz") weiterleitet.
