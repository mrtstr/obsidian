[[py reduce]] applies [[py function]] of two arguments cumulatively to the items of iterable, from left to right, so as to reduce the iterable to a single value.

```python
from functools import reduce

reduce(
	lambda x, y: x + y/2,
	[1, 2, 3, 4]
)
# 5.5
```
