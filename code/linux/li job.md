## [[li job]]
- a [[li job]] is a [[li process]] managed by a [[li shell]] session that is not terminated
#### states
1) running in the background 
2) running in the foreground
3) stopped

### managing jobs
```
jobs           # list running jobs
command &      # start a process in the background
fg %<job id>   # bring background job to foreground
bg %<job id>   # retstart stopped job in the background
kill %<job id> # kill job
ctrl-Z         # stop currently running proces
ctrl-C         # kill currently running proces
```
