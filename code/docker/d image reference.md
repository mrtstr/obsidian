## [[d image reference]]
- full name of a [[d image]]
- contains [[d image repository]]
- a namespace (often app name)
- and a tag (version)

`<docker-registry-hostname>/<repository-name>:<tag>`

- the only required part is the repository-name
- default `docker-registry-hostname = docker.io`
- default `tag = latest`
`docker pull python` → `docker pull docker.io/python:latest`
