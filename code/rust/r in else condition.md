### conditions
- standard if else condition

```rust
let x = 5;

if x > 0 {
    println!("positive");
} else if x < 0 {
    println!("negative");
} else {
    println!("zero");
}
```

- conditional assignments

```rust
let abs = if x >= 0 { x } else { -x };
```
