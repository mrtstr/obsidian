## ecosystem
- most important [[r crate]] for [[rust]]

### serde
- for serializing and deserializing JSON, YAML, TOML
### datafusion
- query engine for [[parquet]] files
- can read and write parquets as hive partitioned dataset
### opendal
- unified [[file system]] interface for different filesytems like [[AWS S3 bucket]] or local...

### object store
- Lightweight abstraction for cloud object stores.
- integrated into Arrow/DataFusion
### crono
- Provides `NaiveDate`, `NaiveTime`, `NaiveDateTime`, and `DateTime<Tz>`.
- Supports parsing/formatting with strftime-like patterns.

### polars
- dataframes

### ndarray
- arrays like [[numpy]]

### ndarray-linalg
- adds linear algebra operations for ndarrays
